3.创建Github Pages

右上角点击 +号,点击New repository

Repository name一定要写成 自己的账户名.github.io,然后勾选上Add a README file

之后点击最下方的Create repository按钮完成Github pages的创建。

5.绑定自己的域名(可选)

我的域名是在腾讯云注册的,所以用腾讯云来操作,
在腾讯云->域名注册->我的域名中找到自己的域名,点击解析

添加如图所示的两条规则,注意记录类型一定要是CNAME

之后在打开github的仓库页面点击Setting->Pages->
在Custom domain中填入自己的域名->勾选Enforce HTTPS->之后等待10分钟

在浏览器中访问自己的域名,显示Hello World!代表成功
